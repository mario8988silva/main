Meta tags

meta charset é definida dentro do head

meta charset define ... meta name define ... content ... initial-scale define a proporção do ecrã do device a ser ocupada; meta name="description" content="pagina de meta tags; Texto que surge abaixo do título do site, no resultado de pesquisas do google, por exemplo" meta name="keywords" content="html, css, jvsscript" meta name="author" content="Mario Silva" meta http-equiv="refresh" content="10" - faz um refresh da página; tag obsoleta; content="x" serve o número de segundos até ao refresh; meta http-equiv="refresh" content="10;url=http://site.com" - leva a outro site ao fim de x segundos; serve para, por exemplo, quando se faz uma revitalização da imagem de marca ou outra similiar que modifique a estrutura ou imagem de um site, deixar o site antigo x segundos até surgir o novo. Serve isto para evitar que o utilizador se sinta confuso ao entrar no site e deparar-se com uma nova imagem; o utilizador não deve ser surpreendido, pela negativa principalmente.
